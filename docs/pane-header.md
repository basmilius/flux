# <flux-pane-header/>

The header of `<flux-pane/>`. Used to display a header.

### Props

| Name    | Type        | Description                   |
|---------|-------------|-------------------------------|
| `icon`  | `IconNames` | The icon used for the header. |
| `title` | `string`    | The title of the header.      |

|---------|---------|
| default | {}      |
