# <flux-button-stack/>

A stack for buttons. Defaults to a horizontal flex container.

### Props

| Name   | Type                         | Description                                    |
|--------|------------------------------|------------------------------------------------|
| `axis` | `'horizontal' \| 'vertical'` | The stack direction.                           |
| `gap`  | `integer`                    | The gap between the elements within the stack. |
