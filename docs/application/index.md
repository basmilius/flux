# Flux Application

Flux Application is a collection of components designed for building modern, route-driven application shells. It provides a complete layout system with a multi-level context menu, sticky top bar, hero header, and right side panel — giving you everything you need to ship a polished application interface without reinventing the layout.

The package builds on Vue Router and uses [named views](https://router.vuejs.org/guide/essentials/named-views.html) to render context-specific menus next to each route, so every level of a nested route tree can contribute its own navigation.
