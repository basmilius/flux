# Flux Visuals

Flux Visuals is a collection of decorative visual effect components. It provides shimmering borders, animated color fields, patterned backdrops and rolling text, helping you add subtle movement and focus to an interface without reaching for a separate animation library.

All effects are purely decorative and `aria-hidden`, and they respect `prefers-reduced-motion`. Because they build on the Flux design tokens, load them alongside `@flux-ui/components`.
